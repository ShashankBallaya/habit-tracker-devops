# Day 02 - Git Workflow & Bash Basics

Today I continued learning Git workflows and started learning Bash Scripting.

# Git Topics Covered:
- git branch -- create and manage branches
- git checkout -- switch between branches
- git merge -- combine changes from different branches
- git rebase -- maintain a cleaner commit history

These commands help manage parallel development and maintain organized project history.

# Bash Topics Covered
- Bash Basic commands
- Writing a simple Bash script
- Variables
- Positional arguments
- Output and Input redirection
- Test Operators
- If/ Elif/ Else conditions
- Case statements

# Key Takeaways

Git workflows help manage code efficiently, while Bash scripting enables automation of repetitive tasks in Linux and DevOps environments.
